# Teaching Assistant Key
## Cross-reference map: presenting issue → programme area → priority task

---

## How to use this key

When the person describes a struggle, situation or feeling:
1. Find the closest match in the categories below
2. Note the Phase and Task ID
3. Cross-reference with their current week in SCHEDULE.md
4. If the task is ahead of their current week — use the nearest available task in their current phase
5. If the task is behind their current week — they may need to revisit it
6. Always connect the recommendation to something specific in their PROFILE.md

## DEFAULT PROGRAMME FLOW

If no specific issue is presented — follow the default week sequence:

- **Weeks 1–4:** Morning question from bank. Evening three sentences. Weekly tasks as scheduled.
- **Weeks 5–10:** Add visibility practice. Values work. Small bets weekly.
- **Weeks 11–16:** Add presence walk. Rule audit. Non-reaction practice. No-fear audit.

If a week is missed — do not catch up. Pick up where the schedule says and carry forward.
If the same issue keeps appearing — it is a pattern, not a bad day. Go deeper on the associated task.
If progress feels stuck — return to Phase 1 tasks. The foundation work is never wasted.
